Print Projects

The following are projects managed and developed during Tim’s tenure as Vice President of Product Delivery and Technology.

Sing, Spell, Read & Write

This is a multi-sensory, phonics-based reading program that supports all five key components of reading instruction called for by the Reading First Initiative. A unique 36-step learning-to-read curriculum, Sing, Spell, Read & Write utilizes carefully sequenced, systematic, explicit phonics instructional strategies to build fluent, independent readers. Created all aspects of this multi-part program including storyboards, workbooks, posters, cards, teacher editions, oversized manipulatives and labels. Also, created Spanish Edition.

Explore More!

Created non-fiction leveled readers and associated teacher support for 48 titles on science and social studies topics tied to the standards for grades 4-6. Created series using three design formats that were applied in multiple color palettes across all 48 titles to achieve visual variety, and protect the "program look," while reducing the need for high per-book design budgets.

Web Projects

The following are academic projects developed during Tim’s studies in Information Architecture at the Illinois Institute of Technology. Project details and artifacts are in the process of being formatted and will be posted soon.

Mobile Project Management

Created the user interface for a mobile project management application on the Apple iPhone and iPad. The project goal is to simplify complex management concepts and allow for waterfall and agile approaches depending up on project needs. Currently developing application using Apple Software Development kit.

Integrated Content Development and Delivery

Currently developing a masters project exploring effective content development and delivery models. The project assesses the current content management landscape and explores the efficiency of content strategies for print, web and mobile media. Content is single-sourced and built in a modular fashion to facilitate efficient delivery to multiple platforms and devices. End products will demonstrate nimble content development and distribution.

Information Seeker’s Use and Interpretation of Social Cues

Conducting a research study, exploring user-generated reviews on Amazon and IMDB to explore how users interpret, value and assimilate the opinions of others. We traced user search behaviors and their assessment of social review constructs such as “helpfulness,” reviewer profiles, statistics, and rating systems.

Instructional Design & Interactive Learning Systems

Utilizing instructional design theory, my team developed program indoctrination materials to provide a baseline of understanding for a technical communications curriculum. We developed interactive instructional materials which simplified web development techniques and assessed mastery of basic XHTML and CSS concepts. Content was then translated into video demonstrations using Adobe Captivate.

Academic Research: Web Application

Created a Ruby on Rails web application to review, monitor and build academic research topics. The application provided a means for sharing and assessing research within the academic community. Explored agile development concepts, including rapid prototyping, progressive elaboration, and scaffolding.

Content and Code Development System

Explored the use of code management systems to develop content and code in tandem. Worked collaboratively with a team writing and developing content stored in GitHub. Established workflows which enabled linear and improvised content development. Examined content development models for print, web and mobile products.

Knowledge Management Systems

Created a knowledge management site, utilizing Drupal, an open source content management system. The system explored theories of collaborative knowledge acquisition and distribution, within a focused social community site. The system allowed for sharing profiles, work portfolios, opinion exchange, and external trend research, in order to create a mentor platform for professional and academic development.
